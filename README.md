# public_issues

공개 정정 요청, 삭제/비노출 요청, 개인정보 문의를 프로젝트별로 접수하기 위한 공개 이슈 저장소입니다.

## 목적

- 코드 저장소와 공개 요청 접수 저장소를 분리합니다.
- 프로젝트가 늘어나도 동일한 이슈 폼 구조로 운영할 수 있게 합니다.
- 사이트에서 유입된 요청은 프로젝트 컨텍스트를 미리 채운 상태로 접수합니다.

## 현재 연결된 프로젝트

- `web_job_information`
  - 사이트: https://web-job-information.pages.dev
  - 코드 저장소: https://github.com/doqltl179/web_job_information

## 접수 유형

- 출처 오류·정정 요청
- 비노출·삭제 요청
- 개인정보·집계 문의

## 공개 저장소 주의사항

- 이 저장소의 이슈는 공개로 접수됩니다.
- 주민등록번호, 전화번호, 이메일, 증빙 문서 원본, 계약서 원본 같은 민감한 개인정보는 본문에 적지 마세요.
- 문제 페이지 URL, 원문 URL, 공개 가능한 설명과 근거 위주로 작성해 주세요.

## 운영 방식

- 이슈 폼은 요청 유형 기준으로 유지합니다.
- 실제 프로젝트 식별은 `project_id`와 관련 메타데이터 필드로 처리합니다.
- 프로젝트별 라우팅과 라벨 정리는 GitHub Actions가 담당합니다.
